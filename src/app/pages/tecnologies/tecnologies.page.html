Technologies and frameworks that I will be using to develop this project;

Html & CSS

Two of the primary technologies for creating Web pages are HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets). For a range of devices, HTML provides the page structure while CSS handles the (visual and auditory) layout. HTML and CSS are the cornerstones of creating Web pages and Web Applications, together with graphics and coding.

JQuery & JQuery mobile

The goal of jQuery is to make using JavaScript on your website simpler. Many typical operations that normally need numerous lines of JavaScript code to complete are wrapped into methods by jQuery so that they can be called with just one line of code. An HTML5-based user interface framework called jQuery Mobile is used to create responsive websites and mobile apps that work on all smartphone, tablet, and desktop devices.

JavaScript

JavaScript enables users to create contemporary web applications that allow direct user interaction without repeatedly refreshing the page. The API frequently uses JavaScript to dynamically modify HTML and CSS to update a user interface. The majority of its applications are web-based.

API

Application Programming Interface is referred to as API. Any software with a specific function is referred to as an application when discussing APIs. Interface can be compared to a service agreement between two programs. This agreement specifies the requests and responses that the two parties will use to communicate.

Visual Studio Code

You may get started coding right now with the help of the free coding editor Visual Studio Code. Without changing editors, use it to code in any programming language. Python, Java, C++, JavaScript, and many other languages are supported by Visual Studio Code.
